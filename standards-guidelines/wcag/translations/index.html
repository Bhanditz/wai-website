WCAG 2 Translations

Summary

This page lists translations of Web Content Accessibility Guidelines (WCAG) 2.1 and 2.0.

Page Contents
  • Types of Translations
  • WCAG 2.1 Completed and Planned Translations
  • WCAG 2.0 Completed and Planned Translations

Types of Translations

W3C provides two types of translations:

  • Authorized Translations follow a formal review process and are endorsed by W3C.
  • Unofficial translations are not reviewed and are not endorsed by W3C.

More Information

For more information on WCAG, see the Web Content Accessibility Guidelines (WCAG) Overview.

Lists of other translsations are available: Translations of all W3C Documents, Translations of WAI Documents, Search for all W3C Translations.

Information about translating W3C resources is provided in W3C Translations and Translating WAI Documents.

All W3C translations, including translations of WCAG 2.0, should be announced to the W3C Translators List (public archive) to facilitate collaboration and to avoid duplication of effort.

For additional information about WCAG translations, contact Shadi Abou-Zahra shadi@w3.org.
