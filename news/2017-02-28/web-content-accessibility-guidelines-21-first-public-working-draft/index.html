Web Content Accessibility Guidelines 2.1 First Public Working Draft

Web Content Accessibility Guidelines (WCAG) 2.1 has been published as a First Public Working Draft. Sites that conform to WCAG 2.1 will also conform to WCAG 2.0, which means they meet the requirements of any policies that reference WCAG 2.0. This first draft includes 28 new Success Criteria, three of which have been formally accepted by the Working Group and the remainder included as proposals to provide an opportunity for early feedback. Further information is available in the blog post: Web Content Accessibility Guidelines 2.1 First Public Working Draft. Please comment by 31 March 2017.
